# Emotional, Human-Like Translation Quality Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make future Chinese→Vietnamese translations emotionally faithful and naturally paced by adding a universal "craft" layer to the translator prompt and a genre-aware "register" layer in the style system.

**Architecture:** Two layers. *Craft* (how to convey emotion/rhythm/voice) is universal and lives in the translator subagent prompt at `.harness/source/agents/translator.md`. *Register* (archaic xianxia vs. modern apocalypse) is per-book and lives in `TranslationStyle` + genre templates under `templates/styles/`. `init-book` stops defaulting every book to archaic xianxia; the agent recommends a genre profile and the user confirms.

**Tech Stack:** Python 3.12, Pydantic v2 (`PersistedModel`), PyYAML, pytest, ruff, `uv`. Harness adapters are generated by `tools/sync_harness_adapters.py` from `.harness/source/**` — never hand-edit generated files under `.claude/`, `.agent/`, `.opencode/`, `.codex/`.

**Spec:** `docs/superpowers/specs/2026-06-15-emotional-translation-quality-design.md`

**Important conventions:**
- All CLI/test commands set UTF-8 and a local uv cache:
  ```powershell
  $env:PYTHONUTF8=1
  $env:UV_CACHE_DIR="$PWD\.uv-cache"
  ```
- Run a single test: `uv run pytest tests/test_styles.py::test_name -v`
- Full gate before finishing: `uv run pytest -q` and `uv run ruff check tools tests src main.py`

```python
def resolve_style_path(
    project_root: Path, custom_style: Path | None = None
) -> Path:
    styles_dir = project_root / "templates" / "styles"
    if custom_style is None:
        return styles_dir / "general.yaml"
    if custom_style.exists():
        return custom_style
    # Bare profile name (e.g. Path("mat_the")) -> templates/styles/<name>.yaml
    named = styles_dir / f"{custom_style.name}.yaml"
    if named.exists():
        return named
    return styles_dir / custom_style.name


def load_selected_style(
    project_root: Path, custom_style_path: Path | None = None
) -> TranslationStyle:
    return load_style(resolve_style_path(project_root, custom_style_path))
```
